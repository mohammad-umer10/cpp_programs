## рЯФє Data Types & Variables

-   **Statically (aesthetically)** = In which we define the data type
    before using the variable.\
-   **Dynamically** = In which we do not explicitly define the data type
    (handled at runtime in some languages).\
-   **Primitive** = Basic data types (int, float, char, etc.).\
-   **Non-Primitive** = User-defined data types (struct, class, array,
    etc.).

------------------------------------------------------------------------

## рЯФє Memory & Storage

-   **Buffer** = A temporary storage area in memory used to hold data.

------------------------------------------------------------------------

## рЯФє Constants & Variables

-   **const** = Value cannot be changed after initialization.\
-   **Variable** = Value can be changed during program execution.

------------------------------------------------------------------------

## рЯФє Scope

-   **Global Variable**
    -   Declared outside any function\
    -   Default value = 0 (if not initialized)
-   **Local Variable**
    -   Declared inside a function or block\
    -   Default value = Garbage value (if not initialized)

------------------------------------------------------------------------

## рЯФє Operators

### вЮ§ Binary Operators

Operators that require two operands:\
- `+` Addition\
- `-` Subtraction\
- `*` Multiplication\
- `/` Division

------------------------------------------------------------------------

### вЮ§ Unary Operators

Operators that work with one operand:\
- `a++` Post Increment\
- `a--` Post Decrement\
- `++a` Pre Increment\
- `--a` Pre Decrement

------------------------------------------------------------------------

### вЮ§ Increment & Decrement

-   **Pre Increment (`++a`)** = Value is increased before use\
-   **Pre Decrement (`--a`)** = Value is decreased before use\
-   **Post Increment (`a++`)** = Value is increased after use\
-   **Post Decrement (`a--`)** = Value is decreased after use

------------------------------------------------------------------------

### вЮ§ Bitwise Operators

-   `&` = AND\
-   `|` = OR\
-   `^` = XOR\
-   `~` = NOT\
-   `<<` = Left Shift\
-   `>>` = Right Shift

------------------------------------------------------------------------

## рЯФє Logical Operators

1.  **Highest** вЖТ `!` (NOT)\
2.  `&&` (AND)\
3.  **Lowest** вЖТ `||` (OR)
